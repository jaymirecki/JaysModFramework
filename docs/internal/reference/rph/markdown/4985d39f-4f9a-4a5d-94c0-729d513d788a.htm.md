Version v0.11.531.2383 was released on Saturday, June 20, 2015 UTC.

Changes in This Release

* RAGEPluginHook.exe will now automatically detect if any files are blocked, and offer to unblock them when it starts (This should make it more convenient, and allow people on Windows 8 to unblock files, since Windows 8 doesn't provide an option to do so).
* Fixed console variable 'WarnOnPluginLoad' being false by default.
* Added "LSPD First Response" version 0.1d Beta as a verified plugin.
* Increased the time plugins are allowed to freeze the game from 1000 ms. to 3000 ms.
* Fixed rendering fibers not being terminated when causing the game to freeze.
* Fixed messages from exceptions being thrown by console commands and variables not being displayed correctly (Always TargetInvocationException).
