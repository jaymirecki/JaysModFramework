Version v0.45.1000.9441 was released on UNKNOWN UTC.

Changes in This Release

* Fixed a bug introduced in v0.44, causing ReloadAllPlugins to not work if not at least one plugin was already loaded.
* Fixed a crash when multiple instances of the same command line option was specified on RagePluginHook.exe.
* Changed the game's default angular velocity limit from one revolution per second, to 7.5 revolutions per second.
* Fixed values of variables passed as ref to DynamicNativeFunction calls not being passed to the natives, effectively making them an out only.
* Added a less eye opening sound for non-error on screen messages.
* Renamed Model::IsElectric to Model::IsElectricVehicle.
* Renamed Model::IsBig to Model::IsBigVehicle.
* Added Ped::CombatTarget.
