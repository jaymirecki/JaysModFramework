Changes in This Release

* Fixed the crash dialog being opened, hidden in the background.
* Added more crash details.
* Fixed the crash dialog not showing a message (as it does when you click "Close"), when clicking the X button to close it.
* Added support for command line options.
* Added command line option "-nodx" to disable DirectX initialization.
