Version v0.15.552.2608 was released on Thursday, June 25, 2015 UTC.

Changes in This Release

* Plugins are now automatically reloaded when modified.
* Added Ped::PlayAmbientSpeech() overload taking the voice to use, the number of the phrase to play, and a speech modifier.
* Fixed World::ConvertWorldPositionToScreenPosition() returning coordinates in percentage instead of pixels.
* Fixed Entity::Metadata always throwing a RuntimeBinderException exception.
* Added Blip constructor overload taking a position and a radius.
* Fixed forms example project not compiling.
* Fixed certain class members being unaccessable in the SDK.
* Added LSPD First Response v0.1e Beta as a verified plugin.
