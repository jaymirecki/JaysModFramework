Changes in This Release

* Updated and added a lot of XML comments.
* Added console command 'Nudge' to push the player's vehicle out of its current predicament.
* Added Ped::IsPainAudioEnabled.
* Renamed Ped::IsStrafing to Ped::IsAiming.
* Fixed PedInventory::EquippedWeapon always returning an invalid weapon.
* Fixed TaskSequence not working.
* Added TaskInvoker::LeaveVehicle() overload with Vehicle parameter.
* Fixed TaskInvoker::TakeCoverFrom() making the Ped shoot at the target from the cover.
