# VehicleWheel Properties

﻿VehicleWheel Properties

|  |  |  |
| --- | --- | --- |
| VehicleWheel Properties | RAGE Plugin Hook Documentation |  |

[This is preliminary documentation and is subject to change.]

The [VehicleWheel](T_Rage_VehicleWheel.htm) type exposes the following members.

Properties

|  | Name | Description |
| --- | --- | --- |
| Public property | [AngleLimit](P_Rage_VehicleWheel_AngleLimit.htm) | Gets or sets the maximum number of degrees this wheel can turn when steering inwards.  This value also affects the maximum angle when turning outwards, however this value will be scaled to be a little larger. |
| Public property | [Health](P_Rage_VehicleWheel_Health.htm) | Gets the health of this wheel. |
| Public property | [IsDriveWheel](P_Rage_VehicleWheel_IsDriveWheel.htm) | Gets or sets a value indicating whether this wheel is a drive wheel. |
| Public property | [IsFacingLeft](P_Rage_VehicleWheel_IsFacingLeft.htm) | Gets or sets a value indicating whether this wheel is facing the left side of the vehicle.  This determines only which way the wheel is facing, not where it's located. |
| Public property | [IsOnFire](P_Rage_VehicleWheel_IsOnFire.htm) | Gets or sets a value indicating whether this wheel is on fire. |
| Public property | [IsSteeringWheel](P_Rage_VehicleWheel_IsSteeringWheel.htm) | Gets or sets a value indicating whether this wheel is used for turning the vehicle. |
| Public property | [IsTouching](P_Rage_VehicleWheel_IsTouching.htm) | Gets a value indicating whether this wheel is currently touching a surface. |
| Public property | [LastContactPoint](P_Rage_VehicleWheel_LastContactPoint.htm) | Gets the position of the point the wheel was last touching. |
| Public property | [SuspensionStrength](P_Rage_VehicleWheel_SuspensionStrength.htm) | Gets or sets the strength of the suspension of this wheel. |
| Public property | [Temperature](P_Rage_VehicleWheel_Temperature.htm) | Gets or sets a value representing the temperature of this wheel.  If the temperature gets too high, the tire will burst. |
| Public property | [TireHealth](P_Rage_VehicleWheel_TireHealth.htm) | Gets the health of the tire of this wheel. |
