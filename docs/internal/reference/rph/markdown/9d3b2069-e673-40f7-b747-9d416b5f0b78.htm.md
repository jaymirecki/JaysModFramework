RAGE Plugin Hook adds a [console](https://en.wikipedia.org/wiki/Console_(video_game_CLI) "Read about command line interfaces in games on Wikipedia.") to the game.

The console allows you to perform various actions like spawning vehicles, load and unload plugins, etc.

It also provides log output.
