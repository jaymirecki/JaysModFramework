Version v0.37.885.6928 was released on UNKNOWN UTC.

Changes in This Release

* Added support for game patch v678.
* Removed support for game patch v617.
* Added counter measures against Rockstar's attempt to keep online vehicles out of singleplayer (Can be disabled under 'Advanced Settings').
* Fixed an access violation in World.IsTimeOfDayFrozen in retail versions of the game, caused by Rockstar's anti-modding protections.
* Fixed Entity::Metadata throwing an exception instead of returning null on non-existent keys.
* Fixed wrong forum link in crash dialog.
* Fixed line under console flickering while dragging it.
* Added native name DRAW\_SCALEFORM\_MOVIE\_3D.
* Added native name CALL\_SCALEFORM\_MOVIE\_METHOD.
* Added native name PLAY\_STREAM\_FROM\_PED.
