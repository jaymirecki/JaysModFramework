[This is preliminary documentation and is subject to change.]

The RAGE engine uses [The International System of Units (SI)](https://en.wikipedia.org/wiki/Common_Language_Infrastructure "Read about The International System of Units on Wikipedia.") for measurements.

Unless otherwise specified, measurements are in meters. Positions are in meters, speeds are in meters per second, etc.
Internally, most angles are stored in radians, however, any angular value in RAGE Plugin Hook is specified in degrees.

This version of the documentation applies to RAGE Plugin Hook v0.56.1131.11508 ALPHA
