# JaysModFramework (JMF)

A singleplayer GTA V modding framework that enables RPG-style game conversions on top of the base game.

The framework acts as an engine layer: it provides systems (entity management, world architecture, save/load, outfit and inventory), and content is defined in data. Plugins built on the framework interact with clean interfaces rather than raw GTA natives.
